-ALORA Skincare Website

A modern, minimal, and aesthetic skincare brand website designed to present products in a clean and visually appealing way.

-About the Project

ALORA is a conceptual skincare brand website focused on elegance, simplicity, and a luxury feel.
The design emphasizes soft tones, clean layouts, and smooth user experience, inspired by modern beauty and skincare brands.

## Features
- Product showcase section  
- Visual slider for featured products  
- Instagram-inspired content section  
- Clean and minimal UI/UX design  
- Contact page with form layout  


-Design & Creativity

All product images used in this project were fully designed and created by me in Canva.
Every detail — from composition, colors, shadows, to overall aesthetic — was carefully crafted to match a premium skincare brand identity.

-Inspiration

I have worked as a Graphic Designer on Upwork, where I collaborated on a similar project for a beauty/makeup brand.
That experience inspired me to create my own concept brand and website — ALORA — where I could fully express my creative vision and style.
